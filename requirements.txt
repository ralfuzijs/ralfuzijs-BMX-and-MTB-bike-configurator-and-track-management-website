# Installation Instructions
# 1. Install Node.js and npm from https://nodejs.org/
# 2. Clone the repository
# 3. Backend setup:
#    cd map-tracks-api
#    npm install
# 4. Frontend setup:
#    cd vue-project
#    npm install
# 5. Start backend server:
#    cd map-tracks-api
#    npm run dev
# 6. Start frontend development server:
#    cd vue-project
#    npm run dev
# 
# Note: SQLite database file will be created automatically on first run.
# No separate database service installation or configuration required.
